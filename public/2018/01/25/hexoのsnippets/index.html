hexoのsnippets

Hexo init hoge

プロジェクト作成

hexo server

サーバ起動

hexo new article_name

記事を作成

hexo new draft draft_name

記事のドラフト（下書き）を作成

hexo publish draft_name

ドラフトを掲載記事にする。(draftsフォルダからpostsフォルダへ移動する)

hexo list hoge

プロジェクト内の色んな一覧がみれる。
page, post, route, tag, categoryのどれか

特にrouteが便利。
